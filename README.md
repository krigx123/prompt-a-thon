# Pixel Pioneer (80th Anniversary Edition)

A retro-style HTML5 side-scrolling platformer with a subtle nod to the 80 years of computing innovation.

## Features

- **Classic Platforming:** Run, jump, and double-jump through endless levels.
- **Dynamic Physics:** Enhanced gravity, acceleration, and jumping mechanics.
- **Obstacles & Enemies:** Avoid hazards, stomp on glitches!
- **Collectibles:** Gather innovation chips and discover hidden power-ups.
- **Responsive:** Playable on desktop and mobile.

## How to Play

- **Desktop:** Use `Arrow Keys` to move, and `Space` or `Up Arrow` to jump. Press `Enter` to start or restart.
- **Mobile:** Use the on-screen touch controls to move and jump.

## Development

This project is built purely with HTML5 Canvas and JavaScript without any external game engines or frameworks.
