# Pixel Pioneer (80th Anniversary Edition)

A retro-style HTML5 side-scrolling platformer crafted with a subtle nod to the IEEE Computer Society's 80 years of computing innovation. Featuring multiple environments, power-ups, and a dynamic zero-gravity space mode!

## 🚀 Key Features

- **Classic Platforming:** Run, jump, and double-jump through endless, procedurally generated levels.
- **Moon Mode (Zero-G):** Press the `G` key to toggle between normal gravity and Moon Mode! Experience deep space zero-gravity physics, maneuver with a jetpack, and dodge alien hazards as a high-tech astronaut.
- **Power-Ups:** 
  - 🍄 **Hardware Upgrade (Mushroom):** Grows your player size and grants an extra hit point!
  - 🔥 **Malware Purger (Fire Flower):** Allows you to shoot bouncing fireballs to purge enemies from a distance.
  - 🧲 **Magnetic Data Chips:** Coins now automatically magnetically gravitate towards you! 
- **Dynamic Enemies:** 
  - *Normal Earth:* Stomp on **Glitch Bugs**, dodge scanning **Drones**, and kick sliding **Security Turtles (Koopas)**!
  - *Moon Mode:* Thrust around undulating **UFOs** and float past heavily spiked **Space Mines**!
- **Responsive:** Fully playable on both Desktop and Mobile!

## 🕹️ Controls

- **Movement:** `Arrow Keys` or `W`, `A`, `S`, `D`
- **Jump / Jetpack Thrust Up:** `Spacebar`, `Up Arrow`, `W`
- **Jetpack Thrust Down (Moon Mode):** `Down Arrow`, `S`
- **Shoot Fireball:** `F` (Requires Malware Purger)
- **Toggle Moon Mode:** `G`
- **Start / Restart Game:** `Enter`

*Note for Mobile: Use the on-screen semi-transparent touch controls to move and jump!*

## ⚙️ Development Highlights

This project was built entirely from scratch with raw HTML5 `<canvas>` and Vanilla JavaScript. It uses a bespoke 2D physics engine running strictly at a smooth cap of 60 FPS, all without any heavy external libraries or game engines.
